This app features:
"Zzzip" the Sleep Detective: A Gemini-powered AI chat companion customized to speak gently to children, asking diagnostic questions in a fun way.
Sleep Journal: A kid-friendly interface to log daily sleep quality.
Dream Weaver: Generates personalized bedtime stories and narrates them using the Gemini Text-to-Speech model.
Parent Dashboard: Uses Gemini to analyze the child's chat and logs to highlight potential sleep disorder patterns (e.g., insomnia, anxiety, apnea).
